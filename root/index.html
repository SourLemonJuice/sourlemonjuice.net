Services reference

Redirect

HTTPS server runs on goto.sourlemonjuice.net with regular port 443.

Like aka.ms or goo.gle, it's a regular short link service. But where would I link to? The fourth dimension space, probably.
Powered with Cloudflare Worker and my Redirect-Worker.

Connectivity test

HTTP/HTTPS server runs on connectivity.sourlemonjuice.net with regular port 80/443.

This domain will always connect to my cloud server instead of CDN.
Maybe some system calls them captive portal, like Android's captive_portal_http_url.
Note: before 2025-02-27, this domain was connection.sourlemonjuice.net

Essentially, pinging this server is possible. But using an HTTP/HTTPS request for URL /generate_204 will given a more realistic result. It's much like connectivitycheck.gstatic.com/generate_204, which is used by a lot of Android devices.

URI Return status Meaning
/generate_204 204 No Content the simplest one, preferred this
/generate_200 200 OK only guarantees that the content is not empty
/NetworkManager 200 OK follow the NetworkManager.conf(5) #CONNECTIVITY_SECTION

DNS recursive server

DNS recursive server runs on dns.sourlemonjuice.net with regular port 53/853/443, means it supports Plain DNS, DNS-over-TLS and DNS-over-HTTPS.
If you don't know that DoT runs on the TCP protocol, please use cloudflare-dns.com thanks.

It's not just a request forward but a recursive server. As for the DNS-over-QUIC, that's not in the plan.
