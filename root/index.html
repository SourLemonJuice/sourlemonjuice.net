Services reference

Redirect

HTTPS server on goto.sourlemonjuice.net, powered with Cloudflare Worker and my Redirect-Worker. Like aka.ms or goo.gle, it's just a regular short link service. But where would I link to? The fourth dimension, probably.

Connection test

HTTP/HTTPS server on connection.sourlemonjuice.net, connect to my cloud server.
Essentially, pinging this server is possible. But using an HTTP/HTTPS request for URL /generate_204 will given a more realistic result. It's much like https://www.google.com/generate_204, which is used by a lot of Android devices.

  • /generate_204 -> return HTTP 204 No Content, preferred this.
  • /generate_200 -> return HTTP 200 OK, only guarantees that the content is not empty.

DNS recursive server

Classic DNS(UDP/TCP) recursive server runs on port 53 of dns.sourlemonjuice.net, it's not a request forward.
